# 🏠 HousePrice AI Predictor

HousePrice AI Predictor is a Machine Learning web application built using Python and Streamlit that predicts house prices based on area demographics and housing features.

---

## ✨ Features

- Real-time house price prediction
- Interactive Streamlit UI
- Multiple Linear Regression model
- Fast and accurate predictions
- Dark-themed modern interface

---

## 🛠️ Tech Stack

- Python
- Streamlit
- Scikit-Learn
- Pandas
- NumPy
- Joblib

---

```bash
streamlit run app.py
```

---

## 📈 Model Performance

- Algorithm: Multiple Linear Regression
- R² Score: 0.918
- Accuracy: 91.8%

---

## 📊 Input Features

- Average Area Income
- House Age
- Number of Rooms
- Number of Bedrooms
- Area Population
